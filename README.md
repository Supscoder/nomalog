# Nomalog Website

A professional, fully responsive multi-page website for Nomalog ICT Solutions and Consulting Services.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Fixed Header**: Professional navigation header with logo on all pages
- **Mobile Menu**: Hamburger dropdown menu for mobile devices
- **Search Bar**: Integrated search functionality
- **Page Transitions**: Smooth rolling circle spinner animation between pages
- **Professional Branding**: Cyan and navy blue color scheme matching the Nomalog logo
- **Multiple Pages**: Home, About, Services, Team, and Contact pages

## Pages

1. **Home (index.html)** - Welcome page with featured infrastructure images
2. **About (about.html)** - Company story, locations, values, and leadership
3. **Services (services.html)** - Comprehensive list of ICT services organized by category
4. **Team (team.html)** - Team members and leadership profiles
5. **Contact (contact.html)** - Contact information and inquiry form

## Files

- `index.html` - Home page
- `about.html` - About page
- `services.html` - Services page
- `team.html` - Team page
- `contact.html` - Contact page
- `style.css` - Main stylesheet with responsive design
- `script.js` - JavaScript for navigation and animations
- `assets/images/logo.svg` - Company logo
- `images/` - Folder for service and team images (see below)

## How to Use

1. Extract the ZIP file to your desired location
2. Create an `images/` folder in the root directory
3. Place all required images in the `images/` folder
4. Open `index.html` in your web browser
5. Navigate between pages using the menu
6. The logo in the header links back to the home page from any page
7. Use the search bar to search for services

## Customization

To customize the website:
- Edit `style.css` to change colors, fonts, or layout
- Edit HTML files to update content
- Replace `assets/images/logo.svg` with your own logo
- Update contact information in all pages as needed
- Add your images to the `images/` folder
